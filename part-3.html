HTML and CSS: Separating content and style

Cascading Style Sheets (CSS) are a powerful way of modifying the look and feel of a website. Although it is tightly integrated with HTML, it uses its own computing language and syntax. This supports a very powerful pattern whereby the content and its logical structure (e.g. which item is a heading, and which one is a paragraph) is separated from its visual aspect (e.g. how big is that heading, how it is aligned, what colour is it?)

Adding styles manually

We’ll first look at how CSS can be created by typing code. This can be done in a basic text editor or in Dreamweaver. We’ll start with a simple example, where we give our “heading 2” <h2> title a green text colour, a light blue background, a dark blue border and we centre it horizontally. The result looks like this:

This is a centered level 2 heading.

We will be using four CSS properties for that:

  • The color property (CSS uses American spelling), whose value will be “green"
  • The background-color property, whose value will be “lightblue”
  • The border property. This one needs not only a colour, but also a weight and a style (which can be solid, dotted, dashed, none, etc.) Here we’re using “5px solid darkblue”
  • The text-align property. We set it to “center”
  • In addition to that, we will add some padding to make sure the text isn’t too close to the border and set it to 0.5em, with “em” being a unit linked to the font size. Please note that you have to use the English decimal sign (.) and not the continental one (,).

So how would we do this?

HTML and CSS working together

There are different ways to style your HTML elements with CSS:

  1. Using the HTML “style” attribute (which is a bad practice that should be avoided).
  2. Using the <style> tag in the head of the HTML file (which is a better practice but still not the best).
  3. Referencing an external CSS file (this is ⭐the best practice⭐ and the only accepted one for the final assignment).

We’ll take a more detailed look at the best practice further below...

Not the best practices
(please avoid!)

index.html
<h2 style="color: green; background-color: lightblue; border: 5px solid dark blue; text-align: center; padding: 0.5em;">This is a centred level 2 heading</h2>

index.html
<!doctype html>
<head>
<title>This is the page's title</title>
<style> .myclass {
color: green;
background-color: lightblue;
border: 5px solid darkblue;
text-align: center;
padding: 0.5em;
} </style>
</head>
<body>
<h2 class="myclass"> This is a centred level 2 heading</h2>
</body>
</html>

⭐Best Practice⭐

Referencing an external CSS file

The best practice (and the only accepted one for the final assignment), is to separate the CSS styles even further and to put them in a separate file, with the CSS extension (styles.css). The benefits are that:

  • Styles from one HTML page can be reused in another one.
  • It uses less bandwidth and disk space because the styles will be only downloaded once for all the pages.
  • It makes it easier for teams of multiple developers to work on different parts of the code (while keeping style consistency across the website).

How to Reference an external CSS file?

The HTML file needs to have a reference to a CSS file using the <link> tag. The “href” attribute uses the same logic as in <a> links or the “src” of an image.

index.html
<!doctype html>
<head>
<title>This is the page's title</title>
<link rel="stylesheet" type="text/css" href="css/styles.css">
</head>
<body>
<h2 class="myclass"> This is a centred level 2 heading</h2>
</body>
</html>
styles.css
.myclass {
color: green;
background-color: lightblue;
border: 5px solid darkblue;
text-align: center;
padding: 0.5em;
}

You know now how to link a CSS document to an HTML document, but how can you link the styles that you declare in your styles.css document to specific elements in HTML page? Keep reading to find out!

Linking CSS styles to specific HTML elements

How can you define to which elements a set of CSS rules apply? This is done through a CSS selector.

There are different types of selectors, in the table you can see the most commonly used and how they work.

While each type of selector is relevant due to its characteristics, class selectors are very special since you can reuse them when you need a specific style (different html elements can use the same class attribute).

Selector types:

  • HTML Element: selects all HTML elements of a given type
  • Class: selects all HTML elements with a given class attribute. Prefix the class name with a dot (.) when using this selector in the CSS file.
  • ID: selects an HTML tag with a given ID attribute. An ID can only be used once in a document. Prefix it with a # when using this selector in the CSS file

Selector types

Selector type CSS code HTML code that gets selected
HTML Element p { color: red; } <p>A paragraph</p>
Class .myclass { color: red; } <h2 class="myclass" >A paragraph</h2>
<p class="myclass other-class" >A paragraph</p>
ID #my-id { color: red; } <p id="my-id" >A paragraph</p>

Practice: add styles manually

  • Open the index.html file that you have been working on or start a new one (it can be in Dreamweaver, Sublime or any other code editor). Make sure there is at least one <p> element with dummy text, also known as “Lorem ipsum”. You can find dummy text online at https://loremipsum.io/
  • Create a new document: the styles.css file and make sure that it is saved inside the same folder as your index.html file and that it is saved with the file extension .css
  • Inside the <head> of your html document include a <link> element with the following attributes and values:
    • rel="stylesheet"
    • type="text/css"
    • href="type here the relative path to your css file"
  • Style three elements in your page are styled, with different styles.
  • Use one class selector and one id selector.

Fonts in CSS

Fonts in CSS are defined with the “font-family” property. If you only use this property, you have to be careful, because you only have access to fonts that are installed. If you want to use additional fonts, you need to embed a web font in your page

Google fonts

Google Fonts is the most popular service for Web Fonts. To use one of their fonts:

  • Go to https://fonts.google.com/ and choose a font you want to use.
  • Click on “Select this style” (do not Download)
  • A sidebar opens where you can see the selected font(s) and instructions
  • Click on “link” and follow the instructions
  • Google gives you one line of code to put in your HTML file, and one to put in your CSS file.

Please note that having Google Fonts activated on your page means that Google can track who is visiting your web site. Nothing is ever completely free!
